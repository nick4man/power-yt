Group-ACL (через oauth2-proxy + group claim в JWT):
─────────────────────────────────────────────────────
  netadmins             → полный admin + GrafanaAdmin
  office-staff          → Editor в Grafana, доступ к office-LAN
  home-power-users      → site-admin home, доступ к media-стек админкам
  family-members        → user в Jellyfin/Seerr/Nextcloud
  family-guests         → guest (read-only) в Jellyfin

Redirect URIs всех клиентов хранятся в auth/realms/{factory-admin,family}.json.
Изменение redirect → правка realm-export → import-realm на следующем deploy.

Установка одного сервиса:
  1. Открыть factory/identity/<service>/*.snippet
  2. Прочитать комменты-инструкции
  3. Заменить REPLACE_WITH_CLIENT_SECRET_FROM_KC реальным secret
     (получить: KC UI → realm → Clients → <id> → Credentials)
  4. Применить snippet на целевом сервисе (config-патч / API-call / occ)
  5. Smoke-test: открыть сервис → SSO-button → авторизация
